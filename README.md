# 🎺 Original Trombones - Product Landing Page

A responsive, modern product landing page for Original Trombones — handcrafted, home-made masterpieces. Built with semantic HTML5 and modern CSS, this page showcases premium product features, embedded video, and clear pricing tiers.

View Demo (Optional: replace with your deployed link)

## 📦 Features

- ⚡ Clean, responsive design using Flexbox and Grid
- 🔥 Font Awesome icons for visual appeal
- 📹 Embedded YouTube video for product showcase
- 💰 Structured pricing section with styled cards
- 📱 Mobile-friendly layout and interactions
- 🎯 Semantic HTML with accessible elements

## 📁 Folder Structure

```
project-root/
│
├── index.html    # Main HTML file
├── styles.css    # CSS styling for layout and components
└── assets/       # (Optional) images or additional media
```

## 🚀 Getting Started

To get this project running locally:

Clone the repo:

```txt
git clone https://github.com/yourusername/product-landing-page.git
cd product-landing-page
```

Open index.html in your browser:

```txt
open index.html
# or just double click the file
Edit as needed: Customize text, styles, or pricing details.
```

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Font Awesome

## ✍️ Customization Tips

- Replace original_trombones.png with your logo.
- Update pricing details and feature lists inside the pricing cards.
- Swap the embedded video URL in the `<iframe>` to showcase your own product demo.
